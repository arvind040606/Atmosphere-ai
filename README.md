🌦️ Atmosphere AI

A modern AI-powered weather forecasting platform built with React, TypeScript, Vite, Tailwind CSS, Framer Motion, and Weather APIs.

Atmosphere AI provides real-time weather information, interactive weather maps, dynamic weather animations, and a premium responsive user interface.

✨ Features
🌍 Smart worldwide location search
📍 Search cities, villages, landmarks & coordinates
🌤️ Real-time weather conditions
📅 Hourly & 10-Day Weather Forecast
🗺️ Interactive Weather Map
🌦️ Dynamic weather animations
🌫️ Air Quality Index (AQI)
🌅 Sunrise & Sunset Information
💨 Wind Speed & Direction
🌙 Dark & Light Theme
📱 Fully Responsive Design

🚀 Getting Started

Clone the repository

git clone https://github.com/arvind040606/Atmosphere-ai.git

Install dependencies

npm install

Create a .env file

VITE_WEATHER_API_KEY=YOUR_WEATHERAPI_KEY
VITE_OWM_API_KEY=YOUR_OPENWEATHERMAP_KEY

Start the development server

npm run dev

Build for production

npm run build
